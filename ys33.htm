The Games Machine


ACE

September 1988
Issue 33

Latest rumour from Amstrad is that there's going to be a 16-bit machine with the Spectrum badge, but won't be called the Spectrum +4. The machine will "likely be a stripped down version of Amstrad's PC 1512 and will run MSDOS software instead of Spectrum games". Rumoured price to be £400 without a monitor.
    Last Ninja II released by System 3, but still no sign of the first Last Ninja...
    The 1982 classic Football Manager gets a sequel.
    Unreleased megagame Battleships returns on a budget label - and gets 6...
    All of Ultimate's games released in one compilation.
    DragonNinja makes an appearance in Slots of Fun.
    YS cover artist Steinar Lund is interviewed.
    A new game called Project 5 is previewed in Future Shocks. The fast action game boasted huge, colourful, well-animated graphics and came in three parts. No one could think of a name for it, even though it had come this far in its development. Then, someone suggested calling the game "Savage"...
