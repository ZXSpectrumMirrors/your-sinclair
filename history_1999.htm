24th September 1999 : Search engine changed from ClaraNET's to AltaVista's since ClaraNET's has been broken for a couple of months now. Also, this site has just been given an Eye Site Award! (Hence the rather large graphic on the front page :))
12th September 1999 : In the final bit of making the front page more dynamic, a Java scroller is added to the top of the page displaying news from this month in the past few years, plus a few quotes about the site. I used the free ProScroll Java applet, adapting the program slightly to handle separate text files for each month, and also so that it always uses a specified image as a separator. At the moment, the only other dynamic features I want to put into the site is automatic changing of graphics so that when Advent/Christmas comes, it'll display the Christmas graphics (once I've created them) automatically.
11th September 1999 : Minor change - the YS banner at the top of site internal pages now changes automatically every month using a bit of JavaScript. Browsers which cannot handle JavaScript shouldn't notice any change :) (Apologies to Netscape 3 users for any mess at the top of the page because the browser has a bug - Netscape 4 doesn't have any problems)

The site's had a bit of a makeover - it hadn't been redesigned since it was first uploaded two years ago, during which time the site has grown beyond all recognition. Also I thought it could look a bit more glossy than it used to. The end result is a better looking site that's easier to navigate. Take a look around this page to see the major feature changes, and also the bits that were chopped off the original cover page to make it a lot smaller and to-the-point. I hope you like the redesign - send me your comments.

28th February 1999 : WOS links fixed.
20th February 1999 : 30,000 UP! I know it's a couple of days late, but just thought I'd make a note of this visitor mile stone, especially as this site seems to have attracted 10,000 visitors since the New Year. Yep, seven weeks to get half the amount of visitors I got the previous 15 months! Also I better tell you that due to the site-moving of World Of Spectrum, the WoS links probably don't work. This site will be updated with the new features in the next few days.
8th February 1999 : And now a minor change - on 19th January, this site was designated a Cool Site in the Netscape Open Directory, hence the icon at the top of the index page.
    Also reprocessed the articles to use the new InfoSeek search engine.
7th February 1999 : A pretty major (for me) format change to the issue summary pages - to take advantage of the new Infoseek search engine on World Of Spectrum, I've changed all the search links so that they use it. Why is this a format change? Well, I can now ditch the form-type buttons and use proper links, eliminating the unsightly blank lines between each entry!!! Many thanks to Martijn van der Heide for introducing this feature for me, now the issue summary pages look as I intended them to look.
    A nice by-product of these changes is that the issue summary pages are now Lynx-friendly, making the whole site reasonably readable for Lynx users.
    This will be the last site update for a while - I'm changing jobs and moving house, so spare time will be a bit short. Still to come are the YS30 screenshots from Paul Jenkinson, plus changing all the search links from the game reviews pages (done the script changes, but an eight-hour long build at the moment cannot be done). After that, who knows?
9th January 1999 : FINALLY!!! The big update I've been promising for absolutely ages! OK, the changes one-by-one:
    112 new articles - Go to this simple index page which lists them all.
    All articles cross-referenced - If a game review that's on this site also has a corresponding re-release or inclusion in a compilation, there'll be a link to it at the end of the review. Take a look at the History In The Making review as an example.
    Compilations and re-releases reviewed - since some original reviews are unscannable, I've tried to fill in the gaps using re-release reviews and compilation reviews, which you can see when you click on the and icons respectively. These reviews are also indexed in the index pages.
    Compilation search links - At the start of every compilation review, there's a set of World Of Spectrum search buttons which allow you to search for snapshots of individual games on the compilation.
    Cover Tape links - When you see the Cover Tape icon (right) in the issue summary pages, clicking on it will take you to the World Of Spectrum's cover tape archive. Good innit?
    Joystick Jugglers updated - I've started to include people in the Joystick Jugglers section who reviewed games for YS before the Jugglers section was established. So far it just covers T'zer, Max Phillips and Sara Biggs. If anyone can point me to the locations of photos for the people who haven't got photos yet I'd be most grateful.
    The Megagames Page - All the YS Megagames in one huge page, complete with review, WoS and screenshot links.
    Screenshots - Paul Jenkinson returns with screenshots for issues 26 to 28.
    Tweaks - Miscellaneous tweaks to the various scripts and formatters.

Current stats for the site: 591 articles (553 reviews, 38 other articles) and 778 screenshots. Comments about the new changes very welcome - email me at nick@ysrnry.co.uk.
