Online Games

There are quite a few interactive things on the site nowadays, so here's a list of what's available.

Who Wants To Be A Squillionaire? : The latest game to be added to the site. Guess Spectrum games from their screenshot.

Spec Trumps : Now you can play that classic card game with Speccy games! This is a phenomenon, easily the most popular thing on the site ever. Nearly ten thousand people played this game in its first week on the site.

The YS Personality Test : Thousands of people have used this to seek light and enlightenment about themselves. They probably didn't find it mind, but they had a good laugh.

The Swedish Speaking Sinclair Spectrum : A Java applet that does exactly what it says on the tin.

YS HyperCapers : An alpha version of a Java applet that I don't think I'll be finishing any time soon. Sort of a cross between YS Capers and HyperSports.

WipeOut! : Rather nifty six-player puzzle game. I've seen players get so competitive that violence usually ensues sooner or later. A Java applet, and not YS-related in any way.

Virtual Competitions

The best YS Megacompos come to life - you too could win a prize, providing you get your entry in to us by the closing date. As the closing dates were in the 1980's, you're just going to have to play these for fun instead! And these games can be played by anyone with a web browser.

The Enormous, Huge, Wonderful, Ace, Triffic, Supersonic, Fab, Over-The-Top, Incredible, Far-Out, Mind-Blowing, Absolutely Ginormous, Mega-Brill YS Compo! : I get the feeling the YS Team were quite pleased with being able to do their first Megacompo...

A Really Rather Big Competition (as competitions go!) : Featuring a pic of T'zer wearing an odd blue-and-white-striped pair of dungarees...

The YS Panto Compo! : Why I did these Virtual Megacompos in the first place - this one is so silly and still quite funny. See the YS team in drag...

YS Goes As Mad As A Hatter! Or A March Hare In May! : Thanks to the cross-referencing scripts used in producing this website, Question 10 is rather easy...
