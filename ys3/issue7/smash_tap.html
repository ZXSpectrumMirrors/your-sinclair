Advanced Lawnmower Simulator 2000
By Cheeky Monkey Studios

Based on elements of the upcoming Ant Attack 2000, this game take lawnmowing simulation to the next stage of evolution. A multiplayer arcade adventure, this demo requires a PC with a 16-bit graphics card. To play in higher resolutions you'll probably need quite a high-end machine, so you'll probably want to play it in low-res.

Read the readme.txt file before playing, and have fun!
