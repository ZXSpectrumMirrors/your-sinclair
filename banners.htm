Banners

If you feel like linking to my website, feel free to use a banner below. Most web browser allow you to save an image by right-clicking it - if not, read your browser's documentation. To make it into a link, add the following bit of HTML to your page:

    <a href="http://www.ysrnry.co.uk"><IMG SRC="banner.jpg"></a>

...and that'll do the trick. By the way, the URL pointed to above will never change, so you can be sure that that link will always work and never need updating.
    Oh, and feel free to trim the banners and resize them to whatever size you want to use.
